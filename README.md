# Hunts Point Haulers

Static marketing site for Hunts Point Haulers LLC — interstate freight carrier.

## Local development

```
npm install
npm start
```

Site serves at http://localhost:3000

## Deployment

Deployed on Railway. `npm start` runs `serve` over the static files — Railway injects `PORT`.
